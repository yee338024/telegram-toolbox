# Прослушивание сообщений

#### 1. Перейдите в `Прослушивание сообщений` - `Управление прослушиванием` для добавления нового слушателя.
![Прослушиватель сообщений](../screenshot/message_listener.png)
#### 2. Добавьте ключевые слова, которые хотите отслеживать.
![Добавить слушатель](../screenshot/add_listener.png)
##### Типы прослушивания
- Точное совпадение: ключевое слово должно полностью совпадать.
- Нечеткое совпадение: ключевое слово может быть частью сообщения.
- Регулярное выражение: используйте регулярные выражения для расширенного поиска.
##### Исключение
При включении исключения сообщения, совпадающие с ключевым словом, будут игнорироваться.
##### Уведомление
При включении уведомлений сообщения, совпадающие с ключевым словом, будут отображаться в виде уведомления Windows (со звуком).
##### Пересылка
При включении пересылки сообщения, совпадающие с ключевым словом, будут автоматически пересылаться в указанный чат.

![Пример пересылки](../screenshot/alert.png)

# Сбор групп

Telegram Toolbox поддерживает автоматический сбор ссылок на группы Telegram из сообщений, помогая пользователям быстро получать и управлять информацией о группах.

## Описание функций
- Автоматическое распознавание ссылок на группы и каналы в сообщениях.
- Одним кликом собирайте информацию о группе, включая название, ID, количество участников и др.

## Как использовать
1. Перейдите на страницу функции `Сбор групп`.
2. Нажмите `Включить`.
3. Система автоматически проанализирует содержимое сообщений, распознает и соберёт ссылки на группы.

![Интерфейс сбора групп](../screenshot/channel_finder.png)

Эта функция подходит для управления сообществами, маркетинговых исследований, интеграции ресурсов и других сценариев, помогая эффективно получать ресурсы групп Telegram.
